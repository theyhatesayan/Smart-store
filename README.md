# 🛍️ TrendyCart - Smart E-Commerce Store

A modern, responsive e-commerce web app built using **HTML, CSS, and Vanilla JavaScript**.
This project demonstrates real-world shopping cart functionality with a clean UI and smooth user experience.

## 📸 Features

* 🛒 Add to Cart functionality
* ➕➖ Increase / Decrease quantity
* ❌ Remove items from cart
* 💾 Cart saved using LocalStorage
* 🔍 Product search functionality
* 💰 Discount price calculation
* 📱 Fully responsive design
* 🎨 Modern glassmorphism UI
* 📦 WhatsApp checkout integration

---

## 🧠 Tech Stack

* HTML5
* CSS3 (Glassmorphism + Responsive Design)
* JavaScript (Vanilla JS)

## ⚙️ How It Works

* Products are stored in `product.js`
* UI is dynamically generated using JavaScript
* Cart data is saved in **LocalStorage**
* Checkout redirects to WhatsApp with order details

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/trendycart.git
```

2. Open the project

```bash
cd trendycart
```

3. Run in browser

* Open `index.html`
